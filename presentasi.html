— Bagian A · Penjelasan Project —

1. GameZone itu apa?

Sistem pemesanan perangkat game

GameZone adalah sistem pemesanan (booking) perangkat game seperti PlayStation dan PC Gaming di sebuah rental/warnet.

Pelanggan bisa:

  • Lihat daftar perangkat tersedia
  • Pesan perangkat untuk waktu tertentu
  • Bayar langsung & dapat tiket (QR)
  • Ikut turnamen & ambil promo
  • Cek riwayat & dompet poin (wallet)

Admin & Staff: kelola perangkat, pengguna, pembayaran, laporan, & pengaturan.

Dibuat dengan PHP Native & MySQL — tanpa framework berat, ringan & mudah dipahami.

2. Struktur Project = Seperti Kantor

Memudahkan memahami fungsi tiap folder
  • Resepsionis → halaman depan (index.php, login.php)

8. Cara Kerja Sistem (Alur)

Dari buka website sampai dapat QR ticket
  1. Pelanggan buka index.php → lihat device
  2. Login → masuk dashboard.php
  3. Pilih Booking: device, waktu, durasi, metode bayar
  4. Sistem validasi → cek device tersedia → hitung total & diskon
  5. Klik bayar → dalam 1 transaksi DB: booking tersimpan, device "dibooking", invoice & payment otomatis dibuat
  6. Pelanggan dapat QR ticket untuk check-in
  7. Staff ubah status via API → device kembali "tersedia" & poin bertambah

9. Teknologi & Keamanan

"Peralatan & Proteksi"
Frontend: HTML, CSS, JS Bootstrap 5 Tailwind Backend: PHP Native 8+ Database: MySQL / MariaDB PDO (aman)
  • Password aman: di-hash, tak disimpan mentah
  • Anti manipulasi: tiap aksi penting pakai CSRF token
  • Hak akses: customer tak bisa buka menu karyawan
  • Transaksi DB: booking pakai BEGIN … COMMIT
  • Kunci ganda turnamen: FOR UPDATE agar kuota tak bentrok
